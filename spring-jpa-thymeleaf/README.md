# 02. Spring Boot + JPA + Thymeleaf Board
**Spring + JPA** 조합을 활용한 게시판입니다.
## 🛠 Tech Stack
- **Backend**: Java 21, Spring Boot 4.0.3
- **Database**: MySQL 8.0
- **ORM/Mapper**: Spring Data JPA (Hibernate)
- **Frontend**: Thymeleaf, HTML/CSS, JavaScript
- **Build Tool**: Gradle
- **IDE** : IntelliJ IDEA
## 💡 Key Learning Points
- **MVC Pattern**: 레이어드 아키텍처(Controller-Service-Repository) 준수
- **JPA**: Entity 기반 ORM, SQL 없이 CRUD 구현
- **Spring Data JPA**: Repository 인터페이스로 기본 CRUD 및 정렬 처리
