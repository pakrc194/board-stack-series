# 🚀 Board Stack Evolution Series

동일한 '게시판' 기능을 다양한 기술 스택으로 구현하며, 각 도구의 특성과 장단점을 깊이 있게 학습하는 프로젝트 시리즈입니다.

## 📌 Project Overview
- **목적**: 기술 스택 변화에 따른 아키텍처 이해 및 생산성 비교
- **주요 기능**: 게시글 CRUD, 페이징, 검색, 사용자 인증 등
- **진행 방식**: 모노레포(Monorepo) 구조로 단계별 기술 스택 확장

## 🗓 Series Roadmap

| 순서 | 기술 스택 (BE / DB / View) | 핵심 키워드 | 상태 |
|:---:|:---|:---|:---:|
| 01 | Spring Boot + MyBatis + Thymeleaf | SQL Mapper, XML | Completed |
| 02 | Spring Boot + JPA + Thymeleaf | ORM, Entity, QueryDSL | In Progress |
| 03 | Python (FastAPI) + MongoDB | NoSQL, Asynchronous | Upcoming |
