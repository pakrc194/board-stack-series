# 01. Spring Boot + MyBatis + Thymeleaf Board

**Spring + MyBatis** 조합을 활용한 게시판입니다.

## 🛠 Tech Stack
- **Backend**: Java 21, Spring Boot 4.0.3
- **Database**: MySQL 8.0
- **ORM/Mapper**: MyBatis
- **Frontend**: Thymeleaf, HTML/CSS, JavaScript
- **Build Tool**: Gradle
- **IDE** : IntelliJ IDEA

## 💡 Key Learning Points
- **MVC Pattern**: 레이어드 아키텍처(Controller-Service-Mapper) 준수
